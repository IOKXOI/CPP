MC Solaar, nique les clones, nique les phonies, 
Nique les crânes qui ne pensent pas, je nique les failles,
Qui flottent comme des crottes sur les réseaux sociaux,
Je nique les clichés, nique les faux degrés,
Et ceux qui ont des œufs dans le pantalon, nique les poules,
Les discours de l'autoroute, les mots des trottoirs,
Nique les comètes, nique les astéroïdes,
Je nique les câlins qui brouillent les capteurs,
Les rêveurs, les marchands, je nique les planètes.
